## What this is

[Next.js Commerce](https://github.com/vercel/commerce) is Vercel's high-performance,
server-rendered ecommerce starter built on the Next.js 14 App Router, React Server
Components, and a Shopify storefront backend. This repository is a point-in-time
copy of that upstream project.

- **Upstream / canonical source:** https://github.com/vercel/commerce
- **Original author / copyright holder:** Vercel, Inc.
- **License:** MIT (see [`license.md`](./license.md) and [`NOTICE`](./NOTICE))
- **Original documentation:** preserved verbatim in [`UPSTREAM_README.md`](./UPSTREAM_README.md)

## Why it's archived here

This account is being tidied up, and repositories that are clones of third-party
open-source work are being labelled honestly rather than presented as original
builds. No attempt has been made to rebrand this code, claim authorship of it, or
pass it off as bespoke work. If you want the real thing — with an active
maintainer, issues, and releases — go straight to the
[upstream repository](https://github.com/vercel/commerce).

## Running it

Setup, environment variables, and the Shopify integration guide live in the
upstream documentation ([`UPSTREAM_README.md`](./UPSTREAM_README.md)). Running the
app requires your own Shopify store and storefront access token; none are included
here (`.env.example` ships with empty placeholders only).

This archived copy is **not deployed** and there is no live demo maintained from
this repository. For a live reference, see Vercel's official demo linked from the
upstream project.
